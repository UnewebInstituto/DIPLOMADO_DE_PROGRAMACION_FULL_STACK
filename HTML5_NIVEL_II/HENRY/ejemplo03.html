Api

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de características y reglas que existen dentro de un programa de software (la aplicación) y que permiten la interacción con él a través del software, a diferencia de una interfaz de usuario humana. La API puede considerarse un simple contrato (la interfaz) entre la aplicación que la ofrece y otros elementos, como software o hardware de terceros.
Referencia MDN

¿Cómo se implementa?

En el desarrollo web, una API es generalmente un conjunto de características de código (por ejemplo, métodos , propiedades , eventos y URL ) que un desarrollador puede usar en sus aplicaciones para interactuar con componentes del navegador web de un usuario, otro software/hardware en la computadora del usuario o sitios web y servicios de terceros.

Por ejemplo:

  • La API getUserMedia se puede utilizar para capturar audio y video de la cámara web de un usuario, que luego está disponible para el desarrollador, por ejemplo, para grabar video y audio, transmitirlo a otro usuario en una llamada de conferencia o capturar imágenes fijas del video.
  • La API de geolocalización se puede utilizar para recuperar información de ubicación de los servicios que el usuario tiene disponibles en su dispositivo (por ejemplo, GPS), que luego se puede utilizar junto con otros servicios, como las API de Google Maps , para trazar la ubicación del usuario en un mapa personalizado y mostrarle qué atracciones turísticas hay en su área.
  • La API de animaciones web se puede utilizar para animar partes de una página web (por ejemplo, para mover o rotar imágenes mediante programación).
