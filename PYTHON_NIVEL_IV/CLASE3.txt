REPASO DEL PROCESO DE MIGRACIÓN:

1-) CREACIÓN DE LA APLICACIÓN QUE TENDRÁ A BBDDD Y DECLARAR EN EL ARCHIVO settings.py
python manage.py startapp agenda

2-) Creación del modelo de entidades de datos. Definir como clases las tablas, por ejemplo (ARCHIVO models.py):

Ejemplo del contenido del archivo: 
./PROFESOR/PROYECTO/AGENDA/models.py
#---------------------------------------------------------
from django.db import models

# Create your models here.
class Contacto(models.Model):
    nombre = models.CharField(max_length=80)
    apellido = models.CharField(max_length=80)
    # EmailField valida el formato y unique=True evita duplicados
    email = models.EmailField(max_length=120, unique=True)
    telefono = models.CharField(max_length=80)
    # TextField se traduce automáticamente a un textarea en formularios
    direccion = models.TextField()

    def __str__(self):
        return f"{self.nombre} {self.apellido}"

#---------------------------------------------------------

3-) VERIFICACIÓN DE LA SINTÁXIS DECLARADA EN EL MODELO.

python manage.py check agenda

4-) CREAR ARCHIVOS DE MIGRACIÓN.
python manage.py makemigrations

Es donde se construyen los archivos con las secuencias enumeradas.

0001_initial.py
0002_....

5-) EJECUTAR EL ARCHIVO SE SECUENCIA ENUMERADA PARA 
PRODUCDIR EL COMANDO SQL.

python manage.py sqlmigrate agenda 0001

6-) CONVERTIR LA DEFINCIÓN EN TABLA FÍSICA:
python manage.py migrate

# Importando la Clase del Modelo
>>>from agenda.models import Contacto

# Se genera una colección de objetos
# contenidos en el modelo
>>> lista_contactos = Contacto.objects.all()

# Aplicar filtro a la colección de objetos
>>> lista_contactos.filter(nombre='YOLANDA')

# Acceder directamente a traves del objetos
>>> Contacto.objects.get(nombre='NELLY')

# Ordenamiento Ascendente o Descendente
# Caso Ascendente 
>>> Contacto.objects.order_by('apellido')

# Caso Descendente
# Signo negativo por delante del nombre
# del campo entre comillas
>>> Contacto.objects.order_by('-apellido')

# Acceso a un registro específico
>>> Contacto.objects.all()[0]
>>> Contacto.objects.all()[1]
>>> Contacto.objects.all()[2]

# Acceso a un registro específico, según
# un orden y número determinado 
# de registros.
# Ascendente
>>> Contacto.objects.order_by('apellido')[0:1]
>>> Contacto.objects.order_by('apellido')[0:2]
>>> Contacto.objects.order_by('apellido')[0:3]

# Descendente
>>> Contacto.objects.order_by('-apellido')[0:1]
>>> Contacto.objects.order_by('-apellido')[0:2]
>>> Contacto.objects.order_by('-apellido')[0:3]

>>> contacto = Contacto.objects.get(nombre='YOLANDA')

>>> contacto

>>> contacto.apellido = 'TORTOZA SUAREZ'

# en multiples registros
